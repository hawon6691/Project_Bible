## 🧾 문서 요약

> `Document/07_code-convention.md`와 `.github/ISSUE_TEMPLATE`를 정리하고, 예시 문서 파일명 규칙을 공통 네이밍 기준에 맞게 맞춘다.

## 🎯 목적 및 배경

> 왜 이 문서가 필요한가요?

- 이슈 문서, 브랜치, 커밋, 파일명 규칙이 타입별로 일관되게 보이도록 기준 문서가 필요했다.
- 앞으로 `Java`, `JavaScript`, `Python`, `PHP`, `Kotlin`, `TypeScript`, `C++`, `C#`, `Go`, `Rust`, `Ruby`, `Lua`까지 같은 규칙으로 확장할 수 있어야 했다.
- `.github/ISSUE_TEMPLATE`도 현재 규칙과 같은 톤으로 맞춰야 이후 이슈 문서 작성이 흔들리지 않는다.
- `Document/07_code-convention` 하위 예시도 실제 파일명 규칙을 바로 보여주는 형태로 정리할 필요가 있었다.
