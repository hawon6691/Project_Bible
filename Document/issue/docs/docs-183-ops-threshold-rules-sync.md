## 🧾 문서 요약

> 어떤 문서인지 한 줄로 설명해주세요.

Ops Dashboard 경보 임계치의 판정 방식과 운영 테스트 절차를 API 명세/운영 런북에 동기화했습니다.

## 🎯 목적 및 배경

> 왜 이 문서가 필요한가요?

- 임계치 환경변수가 추가되었지만 판정식(`>=`)과 비활성화 방식(`<=0`)이 문서에 명확히 정의되지 않았음
- 운영자가 임계치 조정 후 회귀 테스트를 어떤 명령으로 확인할지 기준 문서가 필요함
